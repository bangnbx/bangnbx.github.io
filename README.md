# bangnbx.github.io

Personal technical blog built with [Jekyll](https://jekyllrb.com/) and the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) theme.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit http://127.0.0.1:4000

## Deployment

Automated via GitHub Actions on push to `main`. See `.github/workflows/pages-deploy.yml`.
